# Netflix Content Analysis

## Project Overview
Analyzed 8,800+ Netflix titles to uncover content trends using Python and Power BI.

## Tools Used
- Python (Pandas) — data cleaning & exploration
- Power BI — interactive dashboard

## Key Insights
- Netflix library is 70% Movies vs 30% TV Shows
- Content additions peaked in 2019 with ~2,000 titles added
- US dominates production with 2,818 titles
- Platform skews mature — TV-MA is the most common rating
- International Movies and Dramas are the top genres
